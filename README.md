# Vitepos Javascript
## Admin Panel JavaScript
### Go to admin-panel directory
```shell
cd admin-panel
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
